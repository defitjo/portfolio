More About Me

I started my coding journey by learning from and contributing to The Odin Project. The curriculum started with Ruby, then HTML, CSS, and JavaScript.

After finishing, I was eager to learn more, making projects to learn some JavaScript frameworks such as Vue JS and React.

From there I learned Python, which helped solidify my foundational knowledge I learned previously. Which led to exciting projects utilizing Django, Pygal, and deep learning concepts as well.

With the onset of the pandemic and industries shifting more towards cloud computing, I wanted to understand it more. I first learned Azure then AWS and became certified in both cloud computing services.
